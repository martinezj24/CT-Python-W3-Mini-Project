Contact Management System
Welcome to the Contact Management System! This is a command-line application that allows users to manage their contacts efficiently and effectively.
Users can add, edit, delete, search for, and display contacts, as well as import and export contacts to and from a text file.

Features
Add Contact: Add a new contact with name, phone number, email address, and additional notes.
Edit Contact: Modify existing contact information
Delete Contact: Remove a contact from the list of contacts.
Search Contact: Search for a contact by email address.
Display Contacts: View a list of all contacts with their details.
Import/Export Contacts: Import contacts from a text file or export contacts to a text file.

How To Use:
Once the application is running, you will be presented with a menu offering various options:

1. Add a new contact
2. Edit an existing contact
3. Delete a contact
4. Search for a contact
5. Display all contacts
6. Log Out
Choose the desired option by entering the corresponding number and follow the prompts to perform the action.
